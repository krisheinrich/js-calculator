JavaScript Calculator

Rules on Input:

  • Use four standard arithmetic operations (+, -, *, /) as well as a carat (^) for exponents
  • Use parentheses to specify the order (PEMDAS order of operations are followed by default)
  • Use the return key as a shortcut for the 'Calc' button if you like
  • The 'Ans' key appends the last result to the input box
  • If a calculation does not go through, check your browser's console for information on what went wrong -- you may have forgotten a closing parenthesis, input a letter, etc.
