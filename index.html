JavaScript Calculator

Rules on Input:

  • Use the four standard arithmetic operations (+, -, *, /) as well as a carat (^) for exponents.
    • -1-1+2*3/4^-0.5 --> 10
  • This calculator input supports nesting of expressions. Use parentheses to group subexpressions and control the order in which they are calculated. Standard order of operations are followed by default.
    • (1+6/3)^(9^(2^-1)) --> 27
  • The multiplication sign may be omitted between a number an opening parenthesis: 6(1-4/3)
  • The ANS button recalls the last calculated result.
  • Use the return key as a shortcut for the GO button.

Other features:

  • Input validation: If your input includes a letter, incorrectly formatted decimals (as in 5.15.4) or mismatched parentheses, or if your answer is undefined, the calculator will display an error message with this information and allow you to make changes to you input.
  • The ANS button will skip over any displayed error messages and recall the last numerical output.
  • All displayed answers will be rounded to at most 10 decimal places.
